## How to run Project

### Clone the Repository
```bash
git clone https://github.com/Uva-nana/qrcodegenerator.git
```

### Install the Packages
```bash
pip install -r requirements.txt
```

### Run the app
```bash
python app.py
```

#### Click the  http://127.0.0.1:5000
